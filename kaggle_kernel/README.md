# Kaggle Notebook Upload

This folder is a Kaggle API upload template for the SST training notebook.

To use it, copy `../kaggle_train_sst_new_code.ipynb` into this folder as
`cs224n_gpt2_sst_new_code.ipynb`, then run:

```bash
kaggle kernels push -p kaggle_kernel
```

You need a Kaggle API token at `~/.kaggle/kaggle.json`.
